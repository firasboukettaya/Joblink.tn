================================================================================
                      API ENDPOINTS
================================================================================

1. GET /api/jobs
   - Retrieve job listings with pagination
   - Filters: search, location, source
   - Response: 309+ jobs

2. GET /api/jobs/:id
   - Get specific job details
   - Response: Full job object

3. GET /api/stats
   - Real-time statistics
   - Jobs by source
   - Top locations
   - Recent scraping logs

4. POST /api/scrape
   - Trigger manual scraping
   - Response: Scraping results

5. GET /api/health
   - Server health check
   - Response: {"success": true}

================================================================================
                      GETTING STARTED
================================================================================

Installation:
  1. cd joblink
  2. pnpm install
  3. Create .env file
  4. pnpm start

Testing:
  1. curl http://localhost:3000/api/health
  2. curl http://localhost:3000/api/stats
  3. curl "http://localhost:3000/api/jobs?limit=5"
