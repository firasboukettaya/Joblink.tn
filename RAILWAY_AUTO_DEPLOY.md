## 🎯 Déployer sur Railway en 3 clics

### Étape 1 : Aller sur Railway

Allez sur : **https://railway.app/dashboard**

### Étape 2 : Créer un nouveau projet

1. Cliquez sur **"New Project"**
2. Sélectionnez **"Deploy from GitHub"**

### Étape 3 : Sélectionner votre repository

1. Cherchez **"Joblink.tn"** (ou "joblink.tn")
2. Cliquez dessus
3. Cliquez sur **"Deploy"**

---

## ⏳ Attendre le déploiement

Railway va :
1. ✅ Cloner votre repository
2. ✅ Installer les dépendances (`pnpm install`)
3. ✅ Démarrer votre application
4. ✅ Créer une URL publique

Cela prend **2-5 minutes**.

---

## 🌐 Obtenir votre URL

Une fois le déploiement terminé :

1. Allez dans votre projet Railway
2. Cliquez sur **"Settings"**
3. Sous **"Domains"**, vous verrez votre URL :

```
https://joblink-production.up.railway.app
```

Ou une URL personnalisée si vous l'avez configurée.

---

## ✅ Tester l'application

```bash
# Remplacez par votre URL Railway
URL="https://joblink-production.up.railway.app"

# Tester l'API
curl $URL/api/health
curl "$URL/api/jobs?limit=5"
curl $URL/api/stats
```
